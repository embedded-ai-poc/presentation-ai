2단계

온톨로지: 지도는 생겼지만, 너무 복잡하다

온톨로지로 연결하다

  • 관계 정의 — "차량은 도로 위를 달린다"
  • 지식 그래프 — 거대한 연결망 형성
  • 복잡성 폭발 — 관리가 어려움

온톨로지란?

데이터 간의 관계와 맥락을 정의하여
컴퓨터가 데이터를 이해할 수 있도록 돕는 '설계도'

도로 객체

위치, 차선, 제한속도

차량 객체

종류, 속도, 위치

신호등 객체

상태, 주기, 위치

교통량 객체

시간대, 밀도

3단계

AI의 등장: 스마트 내비게이션

AI가 최적 경로를 안내

  • 패턴 인식 — 자동으로 관계 추론
  • 실시간 반영 — 데이터 변화 즉시 적용
  • 복잡성 해소 — 사람이 관리 안 해도 됨

즉시 최적 경로
