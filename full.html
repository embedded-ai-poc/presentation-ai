연습하기 좋은 저장소

  • FreeRTOS — RTOS 기본, 구조 명확
  • littlefs — 임베디드 파일시스템, 작고 명확
  • TinyUSB — USB 스택, 모듈화 잘됨

보안 주의사항

  • API 키 — 절대 코드에 하드코딩 금지
  • 민감 데이터 — 고객 정보, 내부 키 프롬프트 금지
  • .gitignore — .env, 키 파일 제외 확인
  • 코드 리뷰 — AI 생성 코드도 반드시 검토
